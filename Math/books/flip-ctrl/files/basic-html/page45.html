                    A signal flow graph is a graphic representation of the simultaneous equations describing a system.
                 It graphically displays the transmission of signals through the system, as does the block diagram.


                 3.10 The Signal - Flow Diagram


                    The block diagram is useful tool for simplifying the representation of a system. When a system
                 has more than one input and one output, the control system and block diagram are more complex. An
                 analysis  can  be  further  simplified  by  using  a  signal  flow  graph,  which  has  the  appearance  of  a
                 simplified block diagram.

                    A signal flow graph is a diagram, which represents a set of simultaneous equations. It consists of a
                 network in which nodes, represented by a small dot, are connected by directed branches, represented
                 by a line with an arrow.
